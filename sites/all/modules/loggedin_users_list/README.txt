
-- SUMMARY --

This module displaying how many numbers of users currently logged-in on the
website site. Administrater can see the details of how many users are
loggedin how many system or browsers with logged in time and hostname/IP 
address details too. Administrator has rights to logout any user from his 
machine/browser anytime.

For getting a list of users list, please visit the admin page:
/admin/config/people/loggedin_users_list
